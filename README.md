# 埼玉西武株式会社 コーポレートサイト

## 概要
埼玉西武株式会社の公式コーポレートサイトです。
「野球チーム育成」「食品販売」「アプリ開発」の3つの事業を軸とする同社のビジョンと、チームメンバーを紹介するシングルページ構成のWebサイトとなっています。

## 使用技術
* **HTML5**
* **CSS3** (Flexbox, CSS Grid)

## ファイル構成
* `index.html` : サイトの構造とコンテンツ（文章・画像）
* `style.css` : サイトのデザイン、レイアウト、レスポンシブ対応

## 開発担当（セクション）
チーム開発における各メンバーの担当箇所は以下の通りです。

* **赤羽 担当**: 
  * メインセクション前半
  * キャッチコピー（Heroセクション）
  * 事業内容セクション（野球チーム育成 / 食品販売 / アプリ開発）
* **新納 担当**: 
  * メインセクション後半
  * メンバー紹介セクション（新納会長 / 小林社長 / 赤羽副社長）
* **共通 / その他**:
  * ヘッダー（ナビゲーション）
  * フッター（お問い合わせ先・コピーライト）

## サイトの機能・デザインの特徴
* **レスポンシブ対応**: スマートフォンやタブレットなど、画面幅が狭いデバイス（768px以下）で閲覧した際に、レイアウトが最適化されるようにメディアクエリを使用しています。
* **可変グリッドレイアウト**: 事業内容とメンバー紹介のカード部分は、CSS Grid（`auto-fit`）を採用し、画面サイズに合わせて1列〜3列に自動で並び替わります。
* **追従型ヘッダー**: ページを下にスクロールしても、ヘッダー（ナビゲーションメニュー）が常に画面上部に固定（Sticky）される仕様です。
* **カスタムアイコン**: メンバー紹介のアイコンには、企業の力強さを表現するライオンのGIFアニメーションを採用しています。

## ローカル環境での起動方法
特別な開発環境（サーバーなど）の構築は不要です。

1. リポジトリをクローン、またはファイルをダウンロードします。
2. `index.html` と `style.css` が同じフォルダ内にあることを確認します。
3. `index.html` をGoogle Chrome、Edge、SafariなどのWebブラウザにドラッグ＆ドロップ、またはダブルクリックして開きます。
